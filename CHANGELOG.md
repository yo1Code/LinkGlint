# 更新日志

本项目使用[语义化版本](https://semver.org/lang/zh-CN/)。

## 3.3.0 — 2026-07-17

- 将状态栏单击菜单升级为卡片式网络管理面板，并修复壁纸导致的透明背景染色问题。
- 网络服务严格按 macOS 网络优先级排列，并在面板中显示优先级编号。
- 状态栏新增实时上传/下载速度，可选显示、单行/双行、Byte/bit 单位及 1/2/5 秒刷新间隔。
- 为主窗口和状态栏面板的网络开关增加清晰的绿色开启状态。
- 新增连接其他 Wi-Fi、重命名网络服务、登录启动和原生“关于 LinkGlint”界面。
- 优化状态栏图标、紧凑文字、快捷方案、权限提示及日常操作入口。

## 3.2.0 — 2026-07-16

### 品牌升级

- 项目由 NetBar 更名为 **LinkGlint**（Link + Glint，连接微光）。
- 在 README 顶部加入应用 Logo、品牌标语与新版界面预览。
- 应用、可执行文件、源码模块、测试模块、构建产物和诊断报告统一使用新名称。
- 保留旧版 Bundle ID、偏好设置和权限助手兼容读取，升级后无需重复配置。

## 3.1.0 — 2026-07-16

### 新增与改进

- 项目以 MIT License 公开发布。
- 主窗口重构为紧凑、简约的原生 macOS 界面。
- 将 DNS、优先级、复制和网络切换收纳到适配器“⋯”菜单。
- 将诊断、报告、用量和方案管理收纳到“工具”菜单。
- 关闭最后一个窗口后自动切换为菜单栏模式并移除 Dock 图标。
- 再次显示主窗口或偏好设置时自动恢复标准应用模式。
- 保留菜单栏实时网络状态、流量采样和定时刷新。

### 验证

- 22 项单元测试通过。
- 通用构建同时包含 `x86_64` 与 `arm64`。
- 已验证窗口关闭后进程继续运行、状态项保留且 Dock 中无应用图标。

## 3.0.0 — 2026-07-16

- 加入受限权限助手，首次配置后日常网络操作无需重复输入密码。
- 加入登录时启动、诊断报告、用量统计和配置方案。
- 加入状态栏网络状态文字与实时刷新。
