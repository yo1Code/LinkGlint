# 更新日志

本项目使用[语义化版本](https://semver.org/lang/zh-CN/)。

## 未发布

- “加入 Wi-Fi”改为可浏览附近网络的次级面板，支持刷新、信号与加密状态提示、已保存密码和手动网络输入。
- 修复新版 macOS 中“连接其他 Wi-Fi”等弹窗的输入框被压缩成窄条的问题，并统一表单的尺寸与对齐方式。

## 3.7.0 — 2026-07-18

- 快捷面板新增最近 60 次采样的实时上下行流量曲线，下载与上传分别使用蓝色、橙色展示。
- 曲线采用对数尺度兼顾低流量细节与突发峰值，速率标签使用固定宽度以保持布局稳定。
- 重构状态栏面板开关交互，修复再次单击可能关闭后立即重开的竞态，并支持点击外部稳定关闭。
- 降低状态刷新期间的主线程工作量，菜单操作期间保持流量定时采样，并停止刷新不可见的曲线视图。
- 修复系统命令大量输出时潜在的管道阻塞，并避免旧刷新错误覆盖后续网络操作状态。
- 权限助手的安装与移除改为后台执行，管理员授权期间界面保持响应并防止重复操作。

## 3.6.0 — 2026-07-18

- 固定双行网速的下载、上传标记位置，数值变化时方向提示不再左右移动。
- 新增蓝橙圆点、彩色方向三角和经典箭头三种标记样式，可在偏好设置中即时切换并持久保存。
- 名称与下载标记统一左对齐，数值和单位保持自然紧凑，完整信息展示更规整。
- 彩色标记适配菜单栏明暗外观，单双行布局均使用一致的颜色、尺寸和视觉间距。

## 3.5.2 — 2026-07-18

- 双行网速改用固定宽度的下载、上传锚定区，数字与单位变化时不再带动状态栏项目晃动。
- 两项网速作为紧凑文字组在固定区域内整体居中，避免稳定宽度造成偏斜或大段空白。

## 3.5.1 — 2026-07-18

- 新安装默认登录后仅驻留菜单栏，并默认使用更节省横向空间的双行网速布局。
- 关闭主窗口时保持状态栏宽度不变，不再短暂插入较长的运行提示文字。
- 将右键菜单的用量、诊断、设置和帮助功能收纳为两个子菜单，缩短主菜单。
- 停用当前连接或关闭正在使用的 Wi‑Fi 前增加确认，降低误断网风险。
- 网络启停即时更新界面，后台操作显示进行中与完成状态，失败时立即回滚界面。
- 自动刷新失败改用非打扰式状态提示，只有用户主动刷新失败时才显示警告。
- 明确用量仅为 LinkGlint 的本地运行记录，并移除快捷面板中的低频“关于”按钮及无入口代码。

## 3.5.0 — 2026-07-18

- 以统一四点间距体系重构快捷面板和主窗口，减少零散边距、圆角及控件尺寸。
- 快捷面板压缩宽度和服务行高，移除重复的当前网络卡片，同时保留居中品牌标题和一键刷新。
- 主窗口默认尺寸收紧，六个网络服务可在默认高度完整展示，最小窗口下自动滚动。
- 非活动服务改为轻量无描边列表，活动服务使用低对比强调色和独立“当前/默认”徽标。
- 流量、服务状态及辅助说明提高对比度，长名称和详情增加完整悬停提示。
- 修复状态面板当前网络颜色分支中的强制解包风险，并补充最小尺寸和真实桌面渲染回归。

## 3.4.1 — 2026-07-18

- 状态栏快捷面板新增独立居中的 LinkGlint 标题栏，当前网络继续显示在下方状态卡片中。
- 切换服务时立即更新面板、服务状态和网络图标，后台刷新再校准真实系统状态。
- 面板打开期间保持文字宽度不变，同时允许网络类型图标即时切换。
- 状态栏复合图像按 SF Symbol 原始比例绘制，修复 Wi‑Fi 图标被横向压缩的问题。

## 3.4.0 — 2026-07-18

- 网速改为统计默认路由接口，避免 VPN 与底层物理网卡重复计数。
- 使用标准 `B/s`、`KB/s`、`Mbps` 等单位，并优化状态栏数字字体和字重。
- 快捷面板打开期间暂停状态栏文字重排，关闭后立即补上最新网速，操作时面板不再移动。
- 新增网络服务优先级编辑器，支持拖拽排序以及上移、下移操作。
- 精简 README，突出下载、核心功能、常用操作和构建方式。

## 3.3.1 — 2026-07-17

- 实时流量采样改用系统接口直接读取，移除每次刷新启动 `netstat` 子进程的开销。
- 网络状态没有变化时跳过菜单、主窗口卡片和自动布局树的重复重建。
- 缓存状态栏图像与面板内容，并仅更新当前可见界面，显著降低空闲 CPU 占用。

## 3.3.0 — 2026-07-17

- 将状态栏单击菜单升级为卡片式网络管理面板，并修复壁纸导致的透明背景染色问题。
- 网络服务严格按 macOS 网络优先级排列，并在面板中显示优先级编号。
- 状态栏新增实时上传/下载速度，可选显示、单行/双行、Byte/bit 单位及 1/2/5 秒刷新间隔。
- 为主窗口和状态栏面板的网络开关增加清晰的绿色开启状态。
- 新增连接其他 Wi-Fi、重命名网络服务、登录启动和原生“关于 LinkGlint”界面。
- 优化状态栏图标、紧凑文字、快捷方案、权限提示及日常操作入口。

## 3.2.0 — 2026-07-16

- 应用、可执行文件、源码模块、测试模块、构建产物和诊断报告统一使用新名称。

## 3.1.0 — 2026-07-16
